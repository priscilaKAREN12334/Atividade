O que é o Visual Studio Code?

O Visual Studio Code (ou VS Code) é, basicamente, um editor de códigos. Ele também é chamado de IDE. Todo programador precisa de um IDE, ou editor de códigos, para escrever o código-fonte dos sistemas que está trabalhando. Por exemplo: se você está começando a criar suas páginas web, você pode utilizar o VS Code para digitar as tags HTML da sua página. O VS Code é totalmente gratuito e foi desenvolvido pela Microsoft.
